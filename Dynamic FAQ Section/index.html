JavaScript is a Just-in-time Compiler Programming Language.
It is a multi-paradigm which means that it supports various programming styles(Object-Oriented Programming, Procedural Programming and Functional Programming.)


CSS Grid is a layout system where it is used to create Responsive Designs using Rows and Columns.
Unlike older methods like floats, CSS Grid gives more control over alignment and adapts well to different screen sizes.


Flexbox, short for Flexible Box Layout, is a one-dimensional layout method for aligning and distributing space among items in a container.
Flexbox eliminates the need for floats or complex positioning, enabling responsive and dynamic layouts.
